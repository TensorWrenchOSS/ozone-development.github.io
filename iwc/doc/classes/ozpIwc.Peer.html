ozpIwc.Peer Class

Defined in: app/js/bus/network/peer.js:2
Module: bus.network.packets
Parent Module: bus.network

The peer handles low-level broadcast communications between multiple browser contexts. Links do the actual work of moving the packet to other browser contexts. The links call receive when they need to deliver a packet to this peer and hook the send event in order to send packets.

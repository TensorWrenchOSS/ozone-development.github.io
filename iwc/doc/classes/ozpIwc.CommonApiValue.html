snapshot

() Object

Defined in app/js/bus/api/commonApiValue.js:195

Generates a point-in-time snapshot of this value that can later be sent to {@link ozpIwc.CommonApiValue#changesSince} to determine the changes made to the value. This value should be considered opaque to consumers.

For API subclasses, the default behavior is to simply call toPacket(). Subclasses can override this, but should likely override {@link ozpIwc.CommonApiValue#changesSince} as well.

The OZONE Platform

(next generation OZONE Widget Framework)

The OZONE Platform (OZP) simplifies application discovery and allows applications to share data using the Inter-Widget Communication (IWC) in-browser communications network.

OZP Center

OZP Center is a marketplace that enables the search and discovery of OZP-enabled widgets and applications.

Demo

OZP HUD

The OZP HUD (Heads Up Display) is a convenient landing page for OZP users showing their favorite OZP widgets and applications. Users can launch their applications in a webtop or in separate browser tabs.

Demo

OZP Webtop

The Webtop allows users to create dashboards with multiple OZP widgets/applications running in a single window. Both grid and desktop layouts are provided.

Demo

OZP Demo Apps

This is a collection of OZP widgets/applications used for demo purposes.

OZP Metrics

OZP Metrics is a web interface to display metrics collected through the use of the OZP Center by an end user with valid certificates.
